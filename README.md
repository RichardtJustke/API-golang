🚀 Go API Studies

Repositório criado para documentar minha jornada estudando desenvolvimento de APIs em Go, explorando desde a base com net/http até frameworks modernos e performáticos como Gin e Fiber.

O objetivo deste projeto é aprender na prática:

construção de APIs REST
arquitetura backend
middlewares
autenticação
concorrência
performance
organização de projetos reais

Durante os estudos, estou desenvolvendo desafios e mini projetos utilizando diferentes abordagens para entender como aplicações Go funcionam em ambientes reais e escaláveis.

📚 Tecnologias estudadas
Go (net/http)
Gin
Fiber
JSON APIs
JWT
Middlewares
REST Architecture
PostgreSQL (futuramente)
Docker (futuramente)
🎯 Objetivo

Evoluir de projetos simples para arquiteturas mais próximas do mercado, entendendo não apenas “como fazer”, mas também como construir APIs performáticas, organizadas e escaláveis.

“Aprender backend é descobrir que o verdadeiro boss final sempre foi o banco de dados.” 😅
